# Aural Studios

This repository contains the source code for the official **Aural Studios website**, hosted as a static site via GitHub Pages with a custom domain.

---

## Live Website

**https://auralstudios.in**  
(Custom domain configured via `CNAME`)

---

The website is intentionally lightweight, distraction-free, and focused on identity rather than volume.

---

## Repository Structure

```text
/
├── index.html               # Main landing page
├── style.css                # Global styles
├── script.js                # JavaScript interactions
├── AuralStudiosLogo.svg     # Primary brand logo (SVG)
├── AuralStudiosLogo.jpg     # Logo fallback / raster asset
